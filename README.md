# subband-audio-coding-fir-filter
Digital Signal Processing project implementing a sub-band audio coding system using a two-band FIR filter bank.  The project performs filtering, downsampling, quantization, and reconstruction of speech signals, and evaluates performance using SNR, MSE, and compression ratio.


This project was developed for the Digital Signal Processing course.

Features:
- FIR filter design
- Sub-band signal processing
- Downsampling and upsampling
- Audio compression
- FFT and spectrogram analysis
- Performance evaluation (SNR, MSE, bitrate)

Tools: Python / MATLAB / DSP libraries
